ABOUT THE PROJECT

Brain Tumor Classification System

Objective

To classify brain MRI scans into Glioma, Meningioma, No Tumor, and Pituitary Tumor categories using deep learning.

Model Architecture

ResNet-18 is used as the base convolutional neural network. CBAM attention is added to help the model focus on important image features.

Supported Classes

  • Glioma
  • Meningioma
  • No Tumor
  • Pituitary Tumor

Technology Stack

  • Python and Flask
  • PyTorch
  • ResNet-18 + CBAM
  • HTML, CSS and JavaScript
